BLOCK 03
Modality-Specific Preprocessing
Normalization · Augmentation · Standardization
Endoscopy — CLAHE + specularity masking + elastic deformation
Color jitter, random flip, PraNet-style augmentation
CT — HU windowing W:400 L:50, 3–5mm slice normalization
Tri-planar reformatting (axial / coronal / sagittal)
Histology — 256×256 tiling at 20× magnification
Macenko / Vahadane stain normalization to template
Clinical text — SNOMED-CT term mapping, negation resolution
Temporal relation extraction, symptom chronology
BLOCK 04
Representation Learning — Low-Data Aware
Pretrained encoders · Self-supervised / weakly supervised feature learning
Endoscopy — DINO-pretrained ViT / ConvNeXt + MAE on unlabeled colonoscopy
Domain: HyperKvasir, Gastrovision SSL pretraining
CT — Universal Model / BioViL + SimCLR / MoCo contrastive on local corpus
Transfers anatomical structure representations
Histology — CTransPath / CONCH / UNI frozen encoders
Freeze backbone; train aggregation head only
Text — BioBERT / ClinicalBERT · XLM-RoBERTa for non-English notes
Multilingual support for endemic region records
BLOCK 05
Modality-Specific Analysis Models
Weakly supervised detection · MIL · Risk scoring
Endoscopy — Weakly Supervised Object Localization
LayerCAM / Token-Cut on ViT · DETR / YOLOv8 with frozen backbone (≥50 bbox)
Orientation-sensitive: transverse vs longitudinal
CT — 2.5D Slice-wise Abnormality Detection
3 adjacent slices as RGB → 2D backbone · Separate heads per finding
Wall thickeningLymphadenopathyLuminal narrowing
Histology — Multiple Instance Learning (ABMIL / CLAM)
WSI as bag · Attention weights provide patch-level localization · CLAM for <50 slides
Clinical Text — Structured Risk Scoring
Rule-based Shen/Korean criteria encoding → fine-tuned classifier residual
⚠️ Key differential: ITB-vs-Crohn's — include Crohn's cases as hard negatives with annotated morphological distinctions during training
